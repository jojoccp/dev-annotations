GIT Commands

Comandos Git

Sair da branch sem commitar

$ 
git reset HEAD
$ 
git clean -d -f
$ 
git checkout .


Visualizar os arquivos que foram atualizanos no ultimo commit

$ 
git log --name-only


Subir projeto para Github

$ 
git init
$ 
git status
$ 
git add .
$ 
git commit -m 'meu primeiro commit'
$ 
git branch -M main (ou master)
$ 
git remote add origin 
link_para_o_seu_repositorio
$ 
git push -u origin main


O erro fatal: refusing to merge unrelated histories

$ 
git pull origin master --allow-unrelated-histories
